# Kohala

AI scaffolting system. The AI's `src` folder.

Drop any AI into this structure and it knows how to orient, navigate,
compose, execute, and multi-instance. Not a scaffolder that generates
projects — this IS the project. The AI works from inside it.

## Targets

- Linux (Wayland)
- Windows 10/11

## Architecture

**Headers declare. Includes compose. Routes navigate. Tables decide.
Generators execute. Instances parallelize.**

Instead of one AI spawning sub-agents, Kohala runs multiple AI instances
in parallel — pseudo-agents that share the same scaffolting but operate
on different tasks concurrently.
